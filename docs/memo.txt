

# ROE:J-Quants
# 時価総額:株探
# PER:株探

# 過去10年の経常利益の前年比上昇率の平均
  * 11年分の決算報告が必要
# 直近３回の四半期報告の売上高と経常利益の前年同期比上昇率
  * 2年分の報告書が必要



* 処理の流れ
  * 銘柄リスト取得（プライム・スタンダード・グロース）
  * 各銘柄の次の「計算項目」を求める
    * 時価総額
    * PER
    * ROE
    * 過去10年利益上昇率平均
    * 過去1年売上高上昇率 直近1~」
    * 過去1年利益上昇率 直近1~4
    * 前回報告日
    * 次回報告日（予想）
  * csv 出力する
    * 出力内容は次の通り
      * 銘柄コード
      * 銘柄名
      * 市場
      * 株価
      * 先に計算した「計算項目」
    * ファイル名は pb_<日付>.csv で出力する

* 留意事項
  * stock_database を使用
  * 株探のサイトは使わない
  * J-Quants の API のみを使用
  * 「計算項目」に関してはすでに実装済みのスクリプトがあるので、それを参照する
  * csv はダブルクオート設定をする


* スコアも出していいかも

* 株価から新高値を更新したかどうかを取得できそう

* 企業名・セクタの表示
* 株価のグラフ表示
* 新高値ブレイクタイミングの表示
